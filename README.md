# NoMeta.az

An educational website teaching developers to avoid "meta questions" in online chats.

**Live Site:** [https://nometa.az](https://nometa.az)

## What is a Meta Question?

A meta question is asking for permission to ask, instead of just asking directly:

- "Can I ask a question?"
- "Is anyone here?"
- "Can someone help me?"

**Instead, just ask your actual question directly.**
