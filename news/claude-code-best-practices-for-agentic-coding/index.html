Claude Code: Agentli kodlaşdırma üçün ən yaxşı təcrübələr

Başlanğıc

Claude Code sizin kod bazanızı oxuyan, faylları redaktə edən, əmrləri icra edən və inkişaf alətlərinizlə inteqrasiya edən agentli kodlaşdırma alətidir. Terminalınızda, IDE-də, masa üstü tətbiqində və brauzerdə istifadə edə bilərsiniz.

İşə başlayın

İşə başlamaq üçün mühitinizi seçin. Əksər platformalar Claude abunəliyi və ya Anthropic Console hesabı tələb edir. Terminal CLI və VS Code həmçinin üçüncü tərəf provayderlərini dəstəkləyir.

Terminal

Birbaşa terminalınızda Claude Code ilə işləmək üçün tam funksional CLI. Faylları redaktə edin, əmrləri icra edin və bütün layihənizi komanda sətrindən idarə edin.

Claude Code-u quraşdırmaq üçün aşağıdakı üsullardan birini istifadə edin:

Nativ Quraşdırma (Tövsiyə edilir)

macOS, Linux, WSL:

curl -fsSL https://claude.ai/install.sh | bash

Windows PowerShell:

irm https://claude.ai/install.ps1 | iex

Windows CMD:

curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd

Nativ quraşdırmalar sizi ən son versiyada saxlamaq üçün avtomatik olaraq arxa planda yenilənir.

Homebrew

brew install --cask claude-code

Homebrew quraşdırmaları avtomatik yenilənmir. Ən son funksiyaları və təhlükəsizlik düzəlişlərini almaq üçün vaxtaşırı brew upgrade claude-code əmrini icra edin.

WinGet

winget install Anthropic.ClaudeCode

WinGet quraşdırmaları avtomatik yenilənmir. Ən son funksiyaları və təhlükəsizlik düzəlişlərini almaq üçün vaxtaşırı winget upgrade Anthropic.ClaudeCode əmrini icra edin.

Sonra istənilən layihədə Claude Code-u başladın:

cd your-project
claude

İlk istifadədə daxil olmağınız tələb olunacaq. Budur, vəssalam! Sürətli Başlanğıc ilə davam edin →

Quraşdırma seçimləri, manual yeniləmələr və ya silmə təlimatları üçün qabaqcıl quraşdırma bölməsinə baxın. Problem yaşasanız, problemlərin həlli bölməsinə müraciət edin.

VS Code

VS Code genişlənməsi birbaşa redaktorunuzda inline diff-lər, @-qeydlər, plan nəzərdən keçirmə və söhbət tarixçəsi təmin edir.

VS Code üçün quraşdırın | Cursor üçün quraşdırın

Və ya Genişlənmələr görünüşündə (Mac-da Cmd+Shift+X, Windows/Linux-da Ctrl+Shift+X) "Claude Code" axtarın.

Quraşdırdıqdan sonra Əmrlər Palitrasını açın (Cmd+Shift+P / Ctrl+Shift+P), "Claude Code" yazın və "Open in New Tab" seçin.

VS Code ilə başlayın →

Masa üstü tətbiqi

IDE və ya terminalınızdan kənarda Claude Code-u işə salmaq üçün müstəqil tətbiq. Diff-ləri vizual olaraq nəzərdən keçirin, bir neçə sessiyanı yan-yana icra edin və bulud sessiyalarını başladın.

Yükləyin və quraşdırın:

  • macOS (Intel və Apple Silicon)
  • Windows (x64)

Quraşdırdıqdan sonra Claude-u işə salın, daxil olun və kodlaşdırmağa başlamaq üçün Code tabına klikləyin.

Masa üstü tətbiqi haqqında ətraflı öyrənin →

Veb

Heç bir lokal quraşdırma olmadan brauzerdə Claude Code-u icra edin. Uzunmüddətli tapşırıqlar başladın və hazır olduqda yoxlayın, lokal olaraq mövcud olmayan repo-lar üzərində işləyin və ya birdən çox tapşırığı paralel icra edin. Masa üstü brauzerlərdə və Claude iOS tətbiqində mövcuddur.

claude.ai/code ünvanında kodlaşdırmağa başlayın.

Nə edə bilərsiniz

Claude Code-u istifadə edə biləcəyiniz bəzi yollar:

Həmişə təxirə saldığınız işləri avtomatlaşdırın

Claude Code gününüzü yeyən yorucu tapşırıqları öhdəsinə götürür: test edilməmiş kod üçün testlər yazmaq, layihə boyu lint xətalarını düzəltmək, merge konfliktlərini həll etmək, asılılıqları yeniləmək və buraxılış qeydlərini yazmaq.

claude "write tests for the auth module, run them, and fix any failures"

Funksiyalar yaradın və baqları düzəldin

Nə istədiyinizi sadə dillə təsvir edin. Claude Code yanaşmanı planlaşdırır, bir neçə faylda kodu yazır və işlədiyini yoxlayır. Baqlar üçün xəta mesajını yapışdırın və ya simptomu təsvir edin. Claude Code problemi kod bazanızda izləyir, əsas səbəbi müəyyən edir və düzəliş tətbiq edir.

Daha çox nümunə üçün ümumi iş axınlarına baxın.

Commit-lər və pull request-lər yaradın

Claude Code birbaşa git ilə işləyir. Dəyişiklikləri stage edir, commit mesajları yazır, branch-lar yaradır və pull request-lər açır.

claude "commit my changes with a descriptive message"
